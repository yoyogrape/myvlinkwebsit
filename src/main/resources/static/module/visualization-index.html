产品简介
业务可视化通过整合业务运行数据与IT健康数据，以可视化的方式呈现业务的运行状态、 实现业务与IT的关联智能分析，挖掘数据背后的商业价值，从而驱动业务的增长与发展。
业务运营决策可视化
基于业务过程与业务数据的动态分析，实现对业务流程的优化改进，实现业务决策的可视化。
业务数据可视化
通过对业务数据的采集、处理、存储形成多样化的可视化图表，实现业务的实时数据可视化。
业务架构可视化
通过IT监控、业务监控，掌握业务数据流向及组件相关性，实现对业务系统架构的可视化。
优势
专业可视化平台
面向行业应用提供丰富的行业模板及图表组件，支持上百种图形组件，多维度指标关联与分析辅助业务决策。
丰富的数据接入方式
依托于维联大数据平台的底层开放式接口及协议适配能力，可实现全平台业务数据的实时接入。支持RESTful接口、数据库、日志、其他协议接口的数据接入。
图形化编辑界面
通过拖拉拽的方式快速完成样式与数据源配置，最大化的降低使用人员的学习成本，轻松快速搭建数据大屏幕。
灵活部署与发布
适配各种拼接大屏幕，支持加密与非加密方式的快速发布，支持独立部署或与其他系统集成化部署。
