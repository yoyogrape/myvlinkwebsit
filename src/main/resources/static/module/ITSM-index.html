产品简介
遵从 ITIL、ISO20000、ITSS 等 IT 服务管理领域最佳实践与标准,提供配置灵活的流程引擎, 将 IT 运维管理工作流程化、电子化、自动化,构建 IT 部门的信息化工作管理平台, 帮助用户实现高效运维, 有效提升运维质量和用户满意度水平。
产品功能
自助服务台
       用户可以通过自助服务向导自助解决故障问题, 并与 IT服务部门沟通及提交服务请求,实时跟踪服务进度, 做出满意度反馈和服务质量评价。有效提高 IT 部门服务支持效率, 保证良好的服务质量和充分的互动沟通。
故障报修与服务请求管理
       通过APP、微信、浏览器等多样化的方式进行故障申报,并与知识库、 服务水平管理相结合。提供服务目录管理,方便用户更快速便捷的申请 IT 服务, 用户能够对服务过程进行跟踪。
值班管理
       值班和巡检是保障IT、安防系统的日常性工作， 通过系统的建设，进一步对值班和巡检工作进行有效管控。
资产配置管理
       资产配置全生命周期管理,掌握当前 IT 基础架构中所有资产组件的最新的、 准确的、全面的和详细的信息,实现对所有资产配置项的有效管理。
项目管理
       完成从立项审批到计划编制、实施、关闭等一系列流程化管理,从而降低实施风险、 减少因意外造成的失 败可能性、有效分工合作、控制整体时间进度以及成本。
