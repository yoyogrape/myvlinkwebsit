产品简介
产品定位于提高企业IT部门的核心竞争力,将“监控、管理、治理” 三方面工作有机融合,为客户IT系统提供智能、高效、简单、一体的智能监控平台, 为企业业务提供强有力的IT支撑和质量保障。

开放性

产品功能
一体化综合监控
       系统支持对网络设备、服务器操作系统、数据库、中间件、 存储等IT基础设施及虚拟化的监控管理。集中化7×24小时不间断的管理整个IT系统, 并且监控管理基于标准化协议模型化设计，具备丰富的扩展能力， 方便进行功能扩展和监控资源规模的扩展。
告警管理
       实时掌握所有IT系统、业务系统的运行状态,一旦发现状态异常可以快速定位问题根源点， 并支持多种通知方式及时通知运维人员，提高系统可用性并快速恢复业务。 告警也可以触发运维工单,规范化、流程化的跟踪故障处理进度，确保问题责任到人。
报表统计分析
       报表统计分析是运维质量的度量依据，报表生成灵活、呈现多样， 能满足用户不断变化的统计需求，报表分析能够有序地展现IT资源所有的管理指标， 生成各种分析报告和图表，全面呈现IT系统的设备资源、告警统计、系统运行状况等， 为故障诊断、领导决策提供科学、可量化的依据。
业务服务管理
       从业务的角度统一展现业务系统结构图，提供了基于稳定、重要、 体验的三维能力矩阵评估模型和业务告警规则定义等功能， 使用户可以直观的查看业务系统与IT资源的关系， 可以直观的查看业务系统的告警状况和故障根源， 以便快速恢复业务系统正常运转。
自动化巡检
       规范化运维工作，提前发现系统问题，系统提供自动化巡检的功能。 支持对巡检计划任务管理、巡检内容自定义设置，支持人工巡检和系统自动巡检两种方式， 包括巡检报告人工填写、自动生成与下载导出功能，导出格式支持Excel、Word、PDF。
